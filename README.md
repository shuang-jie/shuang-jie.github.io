# Shuangjie Zhang - Academic Website

This repository contains the source code for my GitHub Pages academic website.

## Website Structure

- `index.md`: Home page
- `research.md`: Research, publications, talks, and software
- `teaching.md`: Teaching and mentoring
- `misc.md`: Personal interests and extra notes
- `another-page.md`: Travel photos page

## Navigation

Top navigation tabs are configured in `_config.yml` under `nav_items`.

## Local Preview (optional)

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://127.0.0.1:4000`.

## Deploy

Push changes to the branch configured in GitHub Pages settings.
